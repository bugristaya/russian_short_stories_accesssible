# Браузерное расширение — ИИ-ассистент корпуса

Расширение является частью репозитория, подготовленного в качестве приложения к выпускной квалификационной работе «Методологии доступного веб-дизайна и их применение к созданию адаптированного литературного ресурса для незрячих пользователей» (НИУ ВШЭ Санкт-Петербург, 2026).

Разработано на основе репозитория [ai-browser-assistant](https://github.com/mngaonkar/ai-browser-assistant); функционал расширения существенно расширен.

---

## Возможности

- **Боковая панель-ассистент** — диалоговое окно выезжает справа поверх страницы, не изменяя её разметку
- **Анализ страницы** — автоматическое краткое изложение содержимого открытой страницы при открытии панели
- **Диалоговый чат** — вопросы и ответы по содержимому страницы с сохранением контекста беседы
- **Голосовой ввод** — распознавание речи на русском языке через Web Speech API (Chromium); отображение статуса разрешения и промежуточного транскрипта
- **Интеграция GigaChat 2 Lite** — языковая модель, обученная на русскоязычных текстах, подключается через прокси-сервер [gpt2giga](https://github.com/ai-forever/gpt2giga)
- **Агентная архитектура LangChain / LangGraph** — классификация типов запросов, управление историей диалога, контекстные потоки



## Параметры модели

- Температура: `0.5`
- Максимальный размер ответа: `2000` токенов
- Контекстное окно страницы: `5000` символов

## Установка

### 1. Получение API-ключа и настройка прокси

1. Получите API-ключ GigaChat на [портале разработчиков Сбера](https://developers.sber.ru/portal/products/gigachat).
2. Разверните прокси-сервер [gpt2giga](https://github.com/ai-forever/gpt2giga), следуя его документации.
3. Создайте файл `.env` в корне проекта по образцу:

```env
GIGACHAT_API_KEY=ваш_ключ
LANGSMITH_API_KEY=ваш_ключ_langsmith   # необязательно
LANGSMITH_PROJECT=browser-assistant    # необязательно
```

### 2. Сборка расширения

```bash
npm install
npm run build
```

Готовые файлы появятся в папке `dist/`.

### 3. Загрузка в Chrome

1. Откройте `chrome://extensions/`
2. Включите **Режим разработчика** (переключатель в правом верхнем углу)
3. Нажмите **Загрузить распакованное расширение**
4. Укажите папку `dist/`


## Настройка через интерфейс расширения

1. Нажмите на значок расширения в панели инструментов Chrome
2. Введите **API-ключ GigaChat** в соответствующее поле
3. При необходимости укажите **Базовый URL API** (адрес прокси-сервера gpt2giga)
4. Опционально добавьте ключ и проект **LangSmith** для трассировки
5. Нажмите **Сохранить**, затем **Проверить**

### Голосовой ввод

1. Нажмите кнопку с иконкой микрофона
2. При первом обращении браузер запросит разрешение на использование микрофона — **состояние запроса отображается в статусной полосе**
3. После получения разрешения начинается запись; **промежуточный транскрипт отображается в реальном времени**
4. После завершения фразы распознанный текст вставляется в поле и автоматически отправляется

> Голосовой ввод работает только в браузерах на основе Chromium.
